HTML & CSS Quick-Reference Guide

HTML Tags

  • <html>: The root element of an HTML document. 
    <html></html>
  • <head>: Contains meta-information about the document. 
    <head></head>
  • <title>: Defines the title of the document. 
    <title>Document Title</title>
  • <link>: Defines the relationship between the current document and an external resource. 
    <link rel="stylesheet" href="styles.css">
  • <body>: Contains the content of the document. 
    <body></body>
  • <h1>: Defines a heading. 
    <h1>Heading</h1>
  • <h2>: Defines a subheading. 
    <h2>Subheading</h2>
  • <ul>: Defines an unordered list. 
    <ul><li>List item</li></ul>
  • <li>: Defines a list item. 
    <li>List item</li>
  • <ol>: Defines an ordered list. 
    <ol><li>List item</li></ol>
  • <hr>: Defines a thematic break in an HTML page (e.g., a shift of topic). 
    <hr>
  • <div>: element is used as a container for other HTML elements, is often used to group sections of a web page together. 
    <div>
    <p>Some text</p>
</div>
  • <p>: Defines a paragraph. 
    <p>This is a paragraph</p>
  • <img>: Inserts an image. 
    <img src="image.jpg" alt="Description">
  • <a>: Creates a hyperlink. 
    <a href="https://example.com">Link</a>
  • <input>: Defines an input control. 
    <input type="text">
  • <label>: Defines a label for an <input> element. 
    <label for="fname">First name:</label><br><input type="text" id="fname" name="fname">
  • <select>: Defines a drop-down list. 
    <select><option value="volvo">Volvo</option></select>
  • <table>: Defines a table. 
    <table><tr><td>Cell</td></tr></table>
  • <th>: Defines a header cell in a table. 
    <th>Header</th>

CSS Properties

  • font-family: Specifies the font for text. 
    body { font-family: Arial, sans-serif; }
  • font-weight: Specifies the weight or thickness of the font. 
    p { font-weight: bold; }
  • font-style: Specifies the style of the font. Common values are normal and italic. 
    p { font-style: italic; }
  • background: Sets all the background properties in one declaration. 
    body { background: #ffffff url('img_tree.png') right top; }
  • background-size: Specifies the size of the background images. 
    body { background-size: 100px 80px; }
  • animation: A property for setting all the animation properties in one declaration. 
    div { animation: mymove 5s infinite; }
  • background-position: Sets the starting position of a background image. 
    body { background-position: right top; }
  • text-align: Specifies the horizontal alignment of text. 
    p { text-align: center; }
  • color: Sets the color of text. 
    p { color: red; }
  • padding-top: Sets the top padding (space) of an element. 
    div { padding-top: 50px; }
  • max-width: Sets the maximum width of an element. 
    div { max-width: 500px; }
  • margin: Sets all the margin properties in one declaration. 
    div { margin: 10px 20px 30px 40px; }
  • padding: Sets all the padding properties in one declaration. 
    div { padding: 10px 20px 30px 40px; }
  • background-color: Sets the background color of an element. 
    body { background-color: lightblue; }
  • border: Sets all the border properties in one declaration. 
    p { border: 1px solid red; }
  • border-radius: Defines the radius of the border of the element. 
    div { border-radius: 25px; }
  • box-shadow: Attaches one or more shadows to an element. 
    div { box-shadow: 0 4px 8px 0 rgba(0, 0, 0, 0.2); }
  • margin-bottom: Sets the bottom margin of an element. 
    p { margin-bottom: 25px; }
